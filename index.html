Canvas.hs, a Haskell library designed to communicate with HTML5/canvas

Canvas.hs

Building UIs in Haskell is never fun, this project aims at fixing this. Just launch it, add some simple calls to your code, and all your graphic primitives will be displayed for all to see!

Install

Installing is easy, get one of the builds as below (Distributed as tarballs) or just clone this repository.

$ git clone git@github.com:CanvasHS/Canvas.hs.git
$ cabal build
$ cabal install
